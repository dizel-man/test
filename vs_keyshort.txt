f12 - передти в класс, метод (содержимое)
alt + f12 - быстрый просмотр класса метода


shift + f12 - (список) показывается где в каких классах он используется
f8 -(найти) перебераем где эти методы используются

ctrl + , - поиск класса файла методом набора текста

ctrl + alt + down arrow - выпадающий список классов непомещающийся в окне
нажимаем как в поиске нужную букву и enter - переходим 

ctrl + shift + > или < - зум как средней мышкой

alt + shift + (мышкой или стрелками) - множественной выделение, замена, добавление public сразу к трем переменным

ctrl + f - поиск
ctrl + shift + f - расширенный поиск

ctrl + shift + Up/Down - переход (найти) выделенное только в данном файле
f8 - в разных файлах

view call hierarchy (ctrl + K(D) -у меня не работает) - просмотреть кто вызывает данный метод 
наподобии shift + f12

множественное копирование ctrl + c
множественное вставка ctrl + shift + v

alt + up/down - перемещение строки вверх вниз
можно выделить несколько строк

shift + del - удалить всю строку 
..

ctrl + space - последнее использование public например


ctrl + alt + space  -  включение выключение автонабора (так же верху есть эконка)

ctrl + . - смарт тэг (вместо лампочки)
создание пока не существующих классов, методов, переменных, using и пр.
быстрая переименовка - меняем название и она предлагает переименовку
подводим к system.using ctrl + . - предлогает удалить неиспользуемые using

или пишем for + tab + tab
class + tab + tab

f9 - поставить брэепоинт (точка откуда будет начата отладка)
